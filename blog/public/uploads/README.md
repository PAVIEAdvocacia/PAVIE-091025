# Uploads do Blog

Esta pasta armazena as imagens dos posts do blog.

## Estrutura

Organize por slug do post:
```
/uploads/
  guia-inventario-internacional/
    cover.jpg
    diagrama.png
  tjrj-inventariante/
    cover.jpg
```

## Boas Práticas

- **Nomes**: usar kebab-case (sem espaços, acentos)
- **Formatos**: JPG (fotos), PNG (diagramas/transparência), WebP (otimizado)
- **Tamanho**: max 1920px largura, otimizado (TinyPNG/ImageOptim)
- **Alt text**: sempre obrigatório no front-matter do post

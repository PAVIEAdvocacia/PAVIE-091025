# ADR-004 — Cloudflare Access no /blog/admin e cache estático
**Status**: Aprovado  
**Data**: 2025-11-03  
**R/A**: R=Security, A=DevOps/Cloudflare  

## Contexto
Admin deve ser privado; conteúdo público deve ser cacheado.

## Decisão
Proteger `/blog/admin` via **Access** com regra por e-mail e MFA. Manter cache estático padrão para público.

## Consequências
+ Camada extra de segurança.  
− Configuração inicial no Zero Trust.

# ADR-003 — Pagefind para busca estática (site + blog)
**Status**: Aprovado  
**Data**: 2025-11-03  
**R/A**: R=FE-Astro, A=SEO Lead  

## Contexto
Busca leve sem servidor para site estático.

## Decisão
Adotar **Pagefind** com `postbuild` após `astro build` e UI acoplável.

## Alternativas
- Algolia (SaaS pago).  
- Lunr/elasticlunr (payload maior).

## Consequências
+ Velocidade, custo zero, privacidade.  
− Index refeito a cada build.

# ADR-006 — GA4 + Consent Mode v2 com eventos customizados
**Status**: Aprovado  
**Data**: 2025-11-03  
**R/A**: R=Data/Analytics, A=PM  

## Contexto
Medição ética e compatível com privacidade.

## Decisão
Ativar Consent Mode v2 e eventos `generate_lead`, `click_whatsapp`, `click_phone`, `download_asset`, `view_faq_item` com parâmetros padrão.

## Consequências
+ Observabilidade sem violar privacidade.  
− Implementar gatilhos de consentimento.

# ADR-010 — Taxonomia de tags e páginas de tag
**Status**: Aprovado  
**Data**: 2025-11-03  
**R/A**: R=SEO Lead, A=Editor-Chefe  

## Contexto
Agrupar conteúdo por áreas do Direito conforme site institucional.

## Decisão
Adotar taxonomia controlada (lista fechada inicial), páginas de tag com descrição e links para serviços.

## Consequências
+ Navegação e SEO temático.  
− Curadoria contínua.

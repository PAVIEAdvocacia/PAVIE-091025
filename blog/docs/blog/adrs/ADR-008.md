# ADR-008 — SEO técnico (sitemap, RSS, hreflang)
**Status**: Aprovado  
**Data**: 2025-11-03  
**R/A**: R=SEO Lead, A=Editor-Chefe

+ Indexação previsível.  
− Manter mapas consistentes.

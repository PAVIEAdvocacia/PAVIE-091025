# ADR-007 — Convenção de slugs, imagens e media folders
**Status**: Aprovado  
**Data**: 2025-11-03  
**R/A**: R=KM/DocOps, A=CMS-Admin  

## Contexto
EvitAR conflitos e facilitar migração/backup.

## Decisão
Usar `YYYY-MM-DD-slug.md`; imagens em `/blog/public/uploads/<slug>/`; `media_folder`/`public_folder` alinhados.

## Consequências
+ Ordem, previsibilidade; menos quebras.  
− Disciplina editorial.
